# Neural Access Protocol 🧠

A Python-based sci-fi authentication system that simulates a secure neural access terminal with multi-layer verification.

---

## 🚀 Features

- Username validation system
- Access key (password) validation
- Neural pattern verification (math-based)
- Security glitch simulation
- User removal from registry on failure
- Terminal-style system output

---

## 🧠 System Overview

Neural Access Protocol is a terminal-based simulation of a futuristic authentication system.

The system is divided into three security layers:

1. **Identity Recognition**  
   Validates if the user exists in the registry  

2. **Access Key Authentication**  
   Ensures the key contains both letters and numbers  

3. **Neural Verification**  
   Challenges the user with math-based checks (multiples of 7)  

Failure at any stage results in access denial and system purge.
